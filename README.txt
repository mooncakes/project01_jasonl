RUN.txt is a plain-text file that shows the console output of the TestCellularData.java file.

TestCellularData.java is a java file that contains a main method that creates an object of the class
CellularData. With it, some countries and data values are hard coded into variables and are used to
run various functions of the CellularData class as well as output an organized table. 

Country.java is a class that contains an object of Country that represents a nation. Each nation has
a name and set of data values. In addition, this object has a method that can be called in order to 
get the number of cellular subscriptions for that country given a start and end year.
